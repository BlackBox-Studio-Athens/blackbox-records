# 08-03 Plan - Add Verified Stripe Webhook Raw-Body Route Contract

## Objective

Add the Worker webhook route that verifies Stripe signatures using raw request body handling.

## Scope Boundary

This commit verifies and acknowledges webhook events. It does not perform paid-order transitions or stock decrement yet.

## Tasks

1. Add `/api/stripe/webhooks` route using `STRIPE_WEBHOOK_SECRET`.
2. Preserve raw body verification before parsing event payloads.
3. Add a narrow allowed-event contract for the checkout-session events needed by v1.
4. Add route tests for valid signature, missing signature, invalid signature, and ignored events.

## Verification

- invalid signatures never reach reconciliation logic
- route acknowledges valid allowed events safely
- public OpenAPI policy is updated if the webhook route is intentionally omitted from shopper docs
- `pnpm test:unit`, `pnpm check`, and `pnpm build` pass

## Explicit Exclusions

- No paid-order mutation
- No stock decrement
- No frontend changes
- No live webhook endpoint provisioning
