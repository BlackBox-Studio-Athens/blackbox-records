# 11-03 Plan - Add Homepage News And Latest Release Feature

## Objective

Replace homepage Latest Releases with a compact News strip and add an editorial latest-release banner above the releases grid.

## Scope

- Homepage shows three latest News items.
- Homepage includes a quiet `/news/` link.
- News remains absent from global header/footer navigation.
- Releases page gets a latest-release feature before the grid.
- Release banner uses existing release catalog sorting and app-shell player conventions.

## Files

- `apps/web/src/pages/index.astro`
- `apps/web/src/pages/releases/index.astro`
- `apps/web/src/content/home/site.json`
- `apps/web/src/content.config.ts`
- `apps/web/src/components/cards/NewsCard.astro`
- `apps/web/src/components/cards/ReleaseCard.astro`
- `apps/web/src/lib/catalog-data.ts`
