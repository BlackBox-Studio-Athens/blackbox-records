# 12-19 Plan - Shell Document Listener Boundary Hardening

## Outcome

Move shell document/window listener attachment and cleanup out of `AppShellRoot` into an internal app-shell helper with
direct characterization coverage.

## Scope

- Characterize listener wiring behavior:
  - document click listener uses capture
  - document pointerover, focusin, and keydown listeners are registered and removed
  - window popstate and blur listeners are registered and removed
  - cleanup removes the same handlers that were registered
- Update `AppShellRoot` to delegate listener attachment while preserving handler logic, routing, overlay, player,
  StoreCart, checkout, Worker, D1, Stripe, BOX NOW, generated API output, `.planning/config.json`, and worktree behavior.
