# 12-26 Plan - Player Iframe Blur Interaction Boundary Hardening

## Outcome

Move player iframe blur interaction detection out of `AppShellRoot` into an internal app-shell helper with direct
characterization coverage.

## Scope

- Characterize player iframe blur interaction behavior:
  - modal-open sessions are marked interacted when the iframe has focus after blur
  - minimized sessions are ignored
  - blur checks are ignored when focus lands elsewhere
- Update `AppShellRoot` to delegate iframe blur interaction detection while preserving player session mutation, iframe
  behavior, modal controls, StoreCart, checkout, Worker, D1, Stripe, BOX NOW, generated API output,
  `.planning/config.json`, and worktree behavior.
