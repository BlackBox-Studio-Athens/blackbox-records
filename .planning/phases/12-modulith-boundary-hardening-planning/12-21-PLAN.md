# 12-21 Plan - Player Modal Focus Boundary Hardening

## Outcome

Move player modal close-button focus scheduling out of `AppShellRoot` into an internal app-shell helper with direct
characterization coverage.

## Scope

- Characterize player modal focus behavior:
  - modal close-button focus is scheduled for the next animation frame
  - missing close buttons are tolerated when the scheduled frame runs
- Update `AppShellRoot` to delegate player modal focus scheduling while preserving player session mutation, iframe
  behavior, modal controls, StoreCart, checkout, Worker, D1, Stripe, BOX NOW, generated API output,
  `.planning/config.json`, and worktree behavior.
