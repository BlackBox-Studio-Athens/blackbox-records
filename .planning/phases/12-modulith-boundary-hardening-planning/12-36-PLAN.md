# 12-36 Plan - App-Shell Internal Folder Organization

## Outcome

Reduce app-shell folder scan cost by moving already-tested app-shell-owned helper/test pairs into behavior folders without
changing shell behavior or public entrypoints.

## Scope

- Keep `AppShell.astro` and `AppShellRoot.tsx` at the app-shell root.
- Leave closed `player` module `player-*` files in place because moving them requires a separate manifest/module
  ownership update.

  - `overlay/`

- Update Phase 12 tracking docs and the app-shell module canvas.
