# 06-06 Plan - Add Release-To-Shop Navigation

## Objective

Let release pages move visitors into the native shop flow using canonical shop identities.

## Tasks

1. Resolve release-derived shop entries by release identity.
2. Update release purchase/navigation affordances to point at native PDPs when mapped.
3. Avoid inventing misleading native shop links for unmapped releases.

## Verification

- mapped releases navigate to native shop PDPs
- unmapped releases do not pretend to have native shop products
- canonical native routes replace raw external links where the mapping exists
